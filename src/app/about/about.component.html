Snapface

Technology Used

Angular 17.3.0, TypeScript 5.4.2

Tutorial Followed

"Get Started with Angular" on OpenClassrooms

Link to the tutorial

Description

Snapface is a modern and interactive web application designed to provide a pleasant experience for viewing and interacting with photos. Developed with Angular framework version 17.3.0 and using TypeScript 5.4.2, Snapface combines performance, responsiveness, and a smooth user interface to allow users to easily navigate through photo collections and "like" them.

Main Features

  • Photo Display: Users can view a gallery of photos organized in a clear and intuitive manner.
  • Photo Likes: Each photo can be "liked" by users, allowing them to show their appreciation.
  • Modern User Interface: The application offers a clean and responsive user interface, ensuring a smooth experience on various devices (desktop, tablets, mobile).
  • Performance and Responsiveness: Thanks to Angular and TypeScript, the application is optimized to offer fast loading times and real-time interactions.

Project Objective

The development of Snapface was undertaken as part of learning Angular. By following the "Get Started with Angular" tutorial on OpenClassrooms, this project aimed to practice the fundamental concepts of Angular, from component structure to service and state management.

Learning and Benefits

  • Mastering Angular: Understanding the basics of the Angular framework, including components, modules, services, and routes.
  • Using TypeScript: Learning to use TypeScript to add strong typing and better error handling to JavaScript code.
  • Developing a SPA (Single Page Application): Designing and implementing a dynamic and responsive web application.

Conclusion

Snapface is the result of practical and in-depth training on Angular, illustrating how to create a performant and enjoyable web application. The "Get Started with Angular" tutorial on OpenClassrooms has been a valuable resource for acquiring the skills needed to complete this project. The application is not only an entertaining tool for users but also a testament to the efficiency and power of the Angular framework combined with TypeScript.
